Arnoldâ€™s Cat Map is a discrete dynamical system known for exhibiting chaotic behavior. It transforms a unit square using a linear transformation followed by a modular reduction, resulting in image scrambling and periodicity.

This project explores:
- The mathematical foundations of the Cat Map
